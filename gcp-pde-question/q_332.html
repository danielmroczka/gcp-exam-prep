Business owners at your company have given you a database of bank transactions. Each row contains the user ID, transaction type, transaction location, and transaction amount. They ask you to investigate what type of machine learning can be applied to the data. Which three machine learning applications can you use? (Choose three.)

  • A. Supervised learning to determine which transactions are most likely to be fraudulent.
  • B. Unsupervised learning to group together transactions with similar characteristics.
  • C. Supervised learning to predict the amount of the next transaction for a given user.
  • D. Reinforcement learning to optimize the interest rates offered to users.
  • E. Reinforcement learning to predict the location of a transaction.
  • F. Unsupervised learning to predict the location of a transaction.
Show Suggested Answer
Suggested Answer: A, B, C

Wyjaśnienie

Correct Answers: A, B, C

A. Supervised learning (classification) can be used for fraud detection if you have historical data with transactions labeled as "fraud" or "not fraud".
B. Unsupervised learning (clustering) is ideal for grouping transactions together based on patterns without needing predefined labels, which can help identify customer segments or unusual behavior.
C. Supervised learning (regression) can be used to predict a continuous value, like the transaction amount, based on historical patterns.

Why other options are incorrect:
D, E, F. Reinforcement learning is about agents taking actions in an environment to maximize a reward, which doesn't directly fit simple transaction prediction tasks. Predicting location (E, F) is typically a classification task (supervised) if labels exist, not unsupervised or reinforcement learning.
