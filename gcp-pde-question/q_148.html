You work for a shipping company that has distribution centers where packages move on delivery lines to route them properly. The company wants to add cameras to the delivery lines to detect and track any visual damage to the packages in transit. You need to create a way to automate the detection of damaged packages and flag them for human review in real time while the packages are in transit. Which solution should you choose?

  • A. Use BigQuery machine learning to be able to train the model at scale, so you can analyze the packages in batches.
  • B. Train an AutoML model on your corpus of images, and build an API around that model to integrate with the package tracking applications. Most Voted
  • C. Use the Cloud Vision API to detect for damage, and raise an alert through Cloud Functions. Integrate the package tracking applications with this function.
  • D. Use TensorFlow to create a model that is trained on your corpus of images. Create a Python notebook in Cloud Datalab that uses this model so you can analyze for damaged packages.

Odpowiedź: B.

Rozwiązanie polegające na wytrenowaniu modelu AutoML na zbiorze obrazów i zbudowaniu API wokół niego jest najbardziej odpowiednie, ponieważ umożliwia integrację z istniejącymi aplikacjami śledzenia przesyłek w czasie rzeczywistym. AutoML automatyzuje proces tworzenia i wdrażania modeli uczenia maszynowego, co jest kluczowe dla szybkiego wdrożenia rozwiązania. API pozwala na łatwe wysyłanie obrazów z kamer i otrzymywanie wyników detekcji uszkodzeń. To podejście zapewnia skalowalność i elastyczność, niezbędną do obsługi dużego wolumenu przesyłek. W ten sposób można automatycznie flagować uszkodzone paczki do weryfikacji przez człowieka w czasie rzeczywistym, co spełnia wymagania firmy.

A. BigQuery ML jest przeznaczone do analizy danych w partiach, a nie do przetwarzania obrazów w czasie rzeczywistym.
C. Cloud Vision API jest dobrym rozwiązaniem, ale może nie być tak precyzyjne jak model wytrenowany na specyficznym zbiorze danych uszkodzeń paczek.
D. TensorFlow wymaga więcej ręcznej pracy i wiedzy specjalistycznej, a Cloud Datalab jest bardziej odpowiedni do eksperymentów niż do wdrożenia produkcyjnego.

*   **Cloud Functions:** https://cloud.google.com/functions

Analiza i Wyjaśnienie

Sugerowana odpowiedź: B

Najbardziej odpowiednią odpowiedzią jest B, czyli wytrenowanie modelu AutoML na zbiorze obrazów i zbudowanie API wokół niego. Dyskusja użytkowników, szczególnie te nowsze wpisy (z ostatnich miesięcy), konsekwentnie potwierdzają tę odpowiedź. Użytkownicy, którzy testowali Cloud Vision API, zauważyli, że nie radzi on sobie z precyzyjnym wykrywaniem uszkodzeń paczek, często myląc je z innymi obiektami. AutoML pozwala na wytrenowanie modelu specyficznie pod kątem rozpoznawania uszkodzeń, co zwiększa dokładność.

Kluczowe argumenty przemawiające za odpowiedzią B:

  • Real-time analysis: AutoML umożliwia analizę obrazów w czasie rzeczywistym, co jest kluczowe dla monitorowania przesyłek w trakcie transportu.
  • Customization and accuracy: Model wytrenowany na specyficznym zbiorze danych uszkodzeń paczek będzie bardziej precyzyjny niż ogólne modele.
  • Integration: API pozwala na łatwą integrację z istniejącymi systemami śledzenia przesyłek.

Dlaczego inne odpowiedzi są nieprawidłowe:

  • A (BigQuery ML): BigQuery ML jest przeznaczone do analizy danych w partiach, a nie do przetwarzania obrazów w czasie rzeczywistym.
  • C (Cloud Vision API): Cloud Vision API jest dobrym narzędziem do ogólnej analizy obrazów, ale nie jest zoptymalizowane do wykrywania specyficznych uszkodzeń paczek. Testy użytkowników potwierdzają, że jego skuteczność w tym zakresie jest niska.
  • D (TensorFlow w Cloud Datalab): TensorFlow wymaga większej wiedzy specjalistycznej i nakładu pracy niż AutoML. Cloud Datalab jest bardziej odpowiedni do eksperymentów niż do wdrożenia produkcyjnego.

Użytkownik juliorevk przeprowadził testy z Cloud Vision API i potwierdził jego niedokładność w wykrywaniu uszkodzeń paczek. Użytkownik MaxNRG szczegółowo wyjaśnił, dlaczego AutoML jest najlepszym rozwiązaniem w tym scenariuszu. Wiele innych komentarzy również wskazuje na odpowiedź B jako najbardziej trafną.
