Question 47 / 361

You want to protect your public-facing application from DDoS attacks and SQL injection. Which service should you use?
  • A. VPC Firewall
  • B. Cloud Armor
  • C. Cloud IAP
  • D. Cloud Endpoints
Show Suggested Answer Hide Answer
Suggested Answer: B

Wyjaśnienie

Odpowiedź: B.

Wyjaśnienie (Dlaczego poprawna):
B. Cloud Armor: Jest to usługa WAF (Web Application Firewall) i ochrony przed DDoS od Google. Oferuje gotowe reguły chroniące przed atakami z listy OWASP Top 10 (w tym SQL Injection, XSS) oraz zaawansowaną ochronę przed atakami wolumetrycznymi DDoS na warstwie 3, 4 i 7.

Dlaczego inne są błędne:
A. VPC Firewall: Działa na warstwie 3/4 (IP/Port). Nie potrafi 'zrozumieć' treści żądania HTTP, więc nie wykryje ataku SQL Injection zaszytego w adresie URL.
C. Cloud IAP: Służy do uwierzytelniania użytkowników, a nie do inspekcji ruchu pod kątem ataków hakerskich.
D. Cloud Endpoints: To brama API (API Gateway) służąca do zarządzania i monitorowania API, nie jest to narzędzie stricte obronne typu WAF.

Konsensus społeczności ExamTopics: 100% na B. Cloud Armor to jedyna poprawna odpowiedź dla ochrony L7 (SQLi/XSS).

Analiza i Podsumowanie Dyskusji (AI)

Analiza i Wyjaśnienie

Sugerowana odpowiedź: B

Głęboka analiza: Cloud Armor współpracuje z Load Balancerem HTTP(S). Ruch jest filtrowany na brzegu sieci Google (Edge), co oznacza, że szkodliwe pakiety nigdy nie docierają do Twojej sieci VPC.
