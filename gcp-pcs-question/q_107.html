Question 107 / 361

Your organization recently deployed a new application on Google Kubernetes Engine. You need to deploy a solution to protect the application. The solution has the following requirements: * Scans must run at least once per week * Must be able to detect cross-site scripting vulnerabilities * Must be able to authenticate using Google accounts Which solution should you use?
  • A. Google Cloud Armor
  • B. Web Security Scanner
  • C. Security Health Analytics
  • D. Container Threat Detection
Show Suggested Answer Hide Answer
Suggested Answer: B

Wyjaśnienie

Odpowiedź: B.

Wyjaśnienie (Dlaczego poprawna):
B. Web Security Scanner: Spełnia wszystkie trzy wymagania: 1. Można harmonogramować skany (raz w tygodniu). 2. Wykrywa XSS (Cross-Site Scripting). 3. Potrafi uwierzytelnić się w aplikacji przy użyciu konta Google, aby przeskanować sekcje dostępne tylko po zalogowaniu.

Dlaczego inne są błędne:
A. Cloud Armor: Chroni przed atakami, ale ich nie 'wykrywa' poprzez skanowanie kodu/aplikacji (nie jest skanerem podatności).
C. Security Health Analytics: Skanuje konfigurację chmury (np. otwarte porty), a nie kod aplikacji pod kątem XSS.
D. Container Threat Detection: Monitoruje pody w czasie rzeczywistym pod kątem podejrzanych procesów (np. reverse shell), nie jest skanerem podatności webowych.

Sugerowana odpowiedź: B

Web Security Scanner to narzędzie typu DAST (Dynamic Application Security Testing) wbudowane w GCP.
