Question 53 / 361

A user needs to view the configuration of a VPC network but should not be able to change anything. Which role should you grant?
  • A. Compute Network Admin
  • B. Compute Network Viewer
  • C. Compute Viewer
  • D. Project Viewer
Show Suggested Answer Hide Answer
Suggested Answer: B

Wyjaśnienie

Odpowiedź: B.

Wyjaśnienie (Dlaczego poprawna):
B. Compute Network Viewer: Rola ta (`roles/compute.networkViewer`) jest precyzyjnie dopasowana do wymogu. Pozwala na podgląd wszystkich zasobów sieciowych (VPC, firewall, trasy), ale nie daje dostępu do podglądu samych maszyn wirtualnych ani żadnych uprawnień edycji.

Dlaczego inne są błędne:
A. Compute Network Admin: Pozwala na modyfikację sieci - naruszenie wymogu 'should not be able to change'.
C. Compute Viewer: Pozwala na podgląd zasobów obliczeniowych (VM), co jest zbyt szerokim uprawnieniem, jeśli użytkownik potrzebuje tylko wglądu w sieć.
D. Project Viewer: Daje dostęp do podglądu WSZYSTKICH usług w projekcie (SQL, Storage itp.) - naruszenie zasady najniższych uprawnień.

Konsensus społeczności ExamTopics: Zdecydowana większość na B jako najbardziej granularną rolę dla administratora sieci bez uprawnień edycji.

Analiza i Podsumowanie Dyskusji (AI)

Analiza i Wyjaśnienie

Sugerowana odpowiedź: B

Głęboka analiza: Wybieranie ról specyficznych dla usługi (np. Network Viewer) zamiast ogólnych (Project Viewer) to kluczowy element zdania egzaminu Professional Security Engineer.
