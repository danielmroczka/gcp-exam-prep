VPC SC for folder with PHI: only US-based personnel, restrict APIs to BigQuery and Cloud Storage only. What to do?
  • A. Scoped access policy, US access level, restrict to all services, VPC accessible only BQ/GCS.
  • B. IAP + US Access level + Restrict Resource Service Usage policy.
  • C. Org-level policy, protect US projects, Geo condition US.
  • D. Cloud Interconnect + VPC firewall + Restrict Resource Service Usage.
Show Suggested Answer Hide Answer
Suggested Answer: A

Wyjaśnienie

Odpowiedź: A.

Wyjaśnienie: Scoped policy (A) izoluje zarządzanie. Funkcja 'VPC accessible services' pozwala na ograniczenie dostępnych API wewnątrz perymetru tylko do wybranych pozycji (GCS, BQ).

Analiza i Podsumowanie Dyskusji (AI)

Analiza i Wyjaśnienie

Sugerowana odpowiedź: A

To rozwiązanie blokuje dostęp do wszystkich innych API Google (np. Pub/Sub, SQL) wewnątrz perymetru.
