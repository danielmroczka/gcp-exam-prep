Question 48 / 361

Applications often require access to `secrets` - small pieces of sensitive data at build or run time. The administrator managing these secrets on GCP wants to keep a track of `who did what, where, and when?` within their GCP projects. Which two log streams would provide the information that the administrator is looking for? (Choose two.)
  • A. Admin Activity logs
  • B. System Event logs
  • C. Data Access logs
  • D. VPC Flow logs
  • E. Agent logs
Show Suggested Answer Hide Answer
Suggested Answer: AC

Wyjaśnienie

Odpowiedź: AC.

Wyjaśnienie (Dlaczego poprawna):
A. Admin Activity logs: Rejestrują operacje zapisu konfiguracji lub metadanych (np. kto stworzył nowy sekret, kto zmienił uprawnienia).
C. Data Access logs: Rejestrują operacje odczytu konfiguracji lub metadanych oraz operacje zapisu i odczytu danych sterowane przez użytkownika (np. kto odczytał wartość sekretu).

Dlaczego inne są błędne:
B. System Event logs: Dotyczą zdarzeń systemowych Google (np. konserwacja maszyn), nie działań użytkownika.
D. VPC Flow logs: Rejestrują ruch sieciowy (IP, porty), a nie konkretne akcje na poziomie API usług (np. Secret Manager).
E. Agent logs: Logi z wewnątrz maszyny VM.
