Question 238 / 361

Your company is concerned about unauthorized parties gaining access to the Google Cloud environment by using a fake login page. You must implement a solution to protect against person-in-the-middle attacks. Which security measure should you use?
  • A. Security key
  • B. Google prompt
  • C. Text message or phone call code
  • D. Google Authenticator application
Show Suggested Answer Hide Answer
Suggested Answer: A

Wyjaśnienie

Odpowiedź: A.

Wyjaśnienie (Dlaczego poprawna):
A. Security Key: Klucze bezpieczeństwa (FIDO2/Titan) są jedyną formą MFA odporną na ataki typu 'person-in-the-middle' (PITM) i phishing. Podczas logowania klucz weryfikuje domenę strony (URL). Jeśli strona jest fałszywa (fake login page), klucz po prostu odmówi podpisania wyzwania, bo domena się nie zgadza.

Dlaczego inne są błędne:
B, C, D: Wszystkie te metody (kody SMS, powiadomienia na telefon, kody z aplikacji) mogą zostać 'przechwycone' przez napastnika na fałszywej stronie i użyte w czasie rzeczywistym do zalogowania się na prawdziwe konto użytkownika.

Analiza i Podsumowanie Dyskusji (AI)

Analiza i Wyjaśnienie

Sugerowana odpowiedź: A

Kryptograficzne powiązanie sesji z domeną (FIDO2) to jedyna skuteczna obrona przed nowoczesnym phishingiem.
