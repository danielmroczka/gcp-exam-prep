Question 127 / 201

You work for a global organization and are running a monolithic application on Compute Engine. You need to select the machine type for the application to use that optimizes CPU utilization by using the fewest number of steps. You want to use historical system metrics to identify the machine type for the application to use. You want to follow Google-recommended practices. What should you do?

  • A. Use the Recommender API and apply the suggested recommendations.
  • B. Create an Agent Policy to automatically install Ops Agent in all VMs.
  • C. Install the Ops Agent in a fleet of VMs by using the gcloud CLI.
  • D. Review the Cloud Monitoring dashboard for the VM and choose the machine type with the lowest CPU utilization.
Show Suggested Answer
Suggested Answer: A 
Odpowiedź: B.

Wyjaśnienie: VPC Service Controls (VPC SC) pozwala na stworzenie obwodu (perimeter), który wymusza, aby zapytania do API usług Google (takich jak Cloud Storage) pochodziły wyłącznie z autoryzowanych sieci VPC. Każda próba połączenia z internetu zostanie odrzucona, nawet przy poprawnych poświadczeniach IAM.

A. IAM kontroluje tożsamość, ale nie potrafi technicznie zablokować dostępu na podstawie źródła sieciowego dla usług PaaS.
C. Firewall VPC chroni maszyny VM, ale nie chroni publicznych adresów IP API Google.
D. Cloud NAT pomaga wychodzić na świat, nie służy do restrykcji dostępu do storage'u.

Analiza i Wyjaśnienie

W Google Cloud VPC Service Controls jest jedynym sposobem na nałożenie restrykcji sieciowych na usługi, które domyślnie są "public-facing" w warstwie API.
