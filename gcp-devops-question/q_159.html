Question 159 / 201

Logging

Your Cloud Run application writes unstructured logs as text strings to Cloud Logging. You want to convert the unstructured logs to JSON-based structured logs. What should you do?

  • A. Modify the application to use Cloud Logging software development kit (SDK), and send log entries with a jsonPayload field.
  • B. Install a Fluent Bit sidecar container, and use a JSON parser.
  • C. Install the log agent in the Cloud Run container image, and use the log agent to forward logs to Cloud Logging.
  • D. Configure the log agent to convert log text payload to JSON payload.
Show Suggested Answer
Suggested Answer: D 
Odpowiedź: B.

Wyjaśnienie: Cloud HSM (Hardware Security Module) to usługa, w której klucze są fizycznie generowane i przechowywane w urządzeniach certyfikowanych zgodnie ze standardem FIPS 140-2 Level 3. Gwarantuje to najwyższy poziom ochrony przed fizyczną ingerencją.

A. Cloud KMS (Software) klucze są w oprogramowaniu (zazwyczaj spełnia tylko FIPS L1).
C. Cloud EKM służy do kluczy poza Google.
D. Secret Manager nie jest systemem HSM do operacji kryptograficznych na kluczach prywatnych.

Analiza i Wyjaśnienie

Wymóg FIPS 140-2 Level 3 jest standardem w bankowości i projektach rządowych. Cloud HSM pozwala na spełnienie tych wymogów bez posiadania własnego sprzętu w serwerowni.
