Question 101 / 201

Logging

You are managing an application that runs in Compute Engine. The application uses a custom HTTP server to expose an API that is accessed by other applications through an internal TCP/UDP load balancer. A firewall rule allows access to the API port from 0.0.0.0/0. You need to configure Cloud Logging to log each IP address that accesses the API by using the fewest number of steps. What should you do first?

  • A. Enable Packet Mirroring on the VPC.
  • B. Install the Ops Agent on the Compute Engine instances.
  • C. Enable logging on the firewall rule.
  • D. Enable VPC Flow Logs on the subnet.
Show Suggested Answer
Suggested Answer: D 
Odpowiedź: A.

Wyjaśnienie: Szyfrowanie deterministyczne (Deterministic Encryption) jest metodą, która dla tej samej wartości wejściowej (np. numeru ID klienta) zawsze zwraca ten sam wynik zaszyfrowany. Jest to kluczowe w scenariuszach analitycznych, gdzie musimy wykonywać operacje typu JOIN między tabelami na danych zaszyfrowanych, zachowując przy tym integralność referencyjną i nie ujawniając danych PII.

B. Hashe z kluczem (HMAC) są jednokierunkowe, co uniemożliwiłoby odwrócenie procesu dla uprawnionych użytkowników.
C. FPE (Format-preserving encryption) zachowuje format (np. numer karty wygląda jak numer karty), ale nie jest standardem dla operacji typu JOIN w bazach danych tak silnie jak deterministyczne AES-SIV.
D. Standardowe hashowanie kryptograficzne bez soli (salt) jest podatne na ataki słownikowe.

Analiza i Wyjaśnienie

W Big Data i systemach bankowych często zachodzi potrzeba łączenia danych z wielu źródeł przy zachowaniu prywatności. Szyfrowanie deterministyczne w Cloud DLP pozwala na bezpieczną analitykę na petabajtach danych.
